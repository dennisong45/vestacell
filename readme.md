# sort

**Description** : It returns a const_reverse_iterator pointing to the last element in the container (i.e., its reverse beginning)..

**Example**:
```cpp

int main () 
{ 
  // declare the vector
  std::vector<int> v = {10, 20, 30, 40, 50}; 
  
  // calling function
  std::v.crbegin();

} 

```
